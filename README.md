# React Router DOM Project

This is a simple React application demonstrating routing using **React Router DOM (v6)**. It includes basic navigation between Home, About, and Contact pages using the `<BrowserRouter>`.

---

## 📁 Project Structure

my-app/
├── public/
│ └── index.html
├── src/
│ ├── components/
│ │ ├── Home.js
│ │ ├── About.js
│ │ ├── Contact.js
│ │ └── Navbar.js
│ ├── App.js
│ └── index.js
├── package.json
└── README.md


## 🚀 Getting Started
1. Install dependencies
```
npm install
```
2. Start the development server
```
npm start
```
This will start the app on http://localhost:3000.

# 📦 Dependencies
->React

->React DOM

->React Router DOM

# 🔗 Navigation
Implemented using react-router-dom@6:

/ – Home page

/about – About page

/contact – Contact page
